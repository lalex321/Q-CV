Q-CV 03.48

Кратко:
Текущий стабильный срез для внутреннего тестирования. В проект положены актуальные рабочие файлы:
- main_03_48.py
- cv_engine_03_48.py
- ai_tasks_03_48.py
- quantori_classic.docx

Что зафиксировано в 03.48:
- Базовый шаблон CV приведён к текущему формату Quantori.
- Порядок core sections соответствует regulations:
  Basics → Summary → Technical Skills → Certifications → Work Experience → Education → Languages.
- Non-core контент должен идти через other_sections.
- Languages рассматривается как отдельная core section.
- Используется актуальный шаблон quantori_classic.docx из проекта.
- Короткий master prompt принят как рабочий baseline вместо старого длинного варианта.
- First-run defaults приведены к текущему пользовательскому сценарию:
  Auto-QA (Audit Only), Generate DOCX = ON, Light theme, основные anonymization flags = ON.
- GitHub import и обычный CV import продолжают дорабатываться на базе этой версии, без смены общей архитектуры tri-file:
  main / cv_engine / ai_tasks.
- Версия предназначена для тестирования коллегами на macOS и Windows.

Важно:
- Source of truth для этой версии — файлы, положенные в проект.
- При дальнейших правках сохранять новые снапшоты по схеме filename_0X_XX + комментарий 0X_XX.txt.
- Старый product doc местами устарел и не полностью отражает текущее состояние 03.48.

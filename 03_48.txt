- First-run defaults aligned with the current user scenario:
  Auto-QA (Audit Only), Generate DOCX = ON, Light theme, main anonymization flags = ON.
- GitHub import and regular CV import continue to be developed on top of this version, without changing the overall tri-file architecture:
  main / cv_engine / ai_tasks.
- This version is intended for testing by colleagues on macOS and Windows.
